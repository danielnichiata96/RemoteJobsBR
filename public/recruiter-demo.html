Portal do Recrutador - Demonstração

Esta página mostra os componentes desenvolvidos para o portal do recrutador. Estes componentes são implementados usando React, TypeScript, Tailwind CSS e apresentam funcionalidades completas para gerenciamento de vagas e candidatos.

Dashboard do Recrutador

O dashboard apresenta estatísticas importantes como vagas ativas, candidaturas recebidas, entrevistas agendadas e um histórico de atividades recentes.

Funcionalidades:
  • Exibição de estatísticas de recrutamento
  • Listagem de atividades recentes com timeline
  • Links rápidos para ações comuns

Gerenciamento de Vagas

Permite que o recrutador visualize, edite e gerencie todas as vagas publicadas pela empresa.

Funcionalidades:
  • Listagem de todas as vagas publicadas
  • Filtragem por status (ativa, encerrada, rascunho)
  • Ações rápidas para editar e encerrar vagas
  • Visualização de candidaturas por vaga

Formulário de Publicação de Vaga

Formulário completo para criar e editar vagas, com validação de campos e suporte para várias informações relevantes.

Funcionalidades:
  • Validação completa de formulário usando Zod e React Hook Form
  • Suporte para informações detalhadas da vaga
  • Seleção de habilidades/tecnologias com tags
  • Configuração de requisitos, responsabilidades e benefícios

Revisão de Candidaturas

Interface para revisar e gerenciar candidaturas recebidas para uma vaga específica.

Funcionalidades:
  • Listagem de candidatos com informações essenciais
  • Filtragem por status de candidatura
  • Modal detalhado com informações completas do candidato
  • Gerenciamento de fluxo de recrutamento (analisar, entrevistar, contratar, rejeitar)
  • Acesso a currículo e carta de apresentação

Interface de Mensagens

Sistema de chat para comunicação direta entre recrutadores e candidatos.

Funcionalidades:
  • Lista de contatos com indicadores de mensagens não lidas
  • Interface de chat completa com histórico de mensagens
  • Suporte para envio de arquivos e anexos
  • Indicadores de status online/offline

Implementação e Integração

Estes componentes foram implementados usando as seguintes tecnologias:

  • React e TypeScript para os componentes de interface
  • Tailwind CSS para estilização
  • React Hook Form e Zod para validação de formulários
  • Componentes de UI do HeroIcons para ícones

Para implementar estes componentes em um projeto real, você precisará:

  1. Conectar os componentes a APIs reais para buscar e salvar dados
  2. Implementar autenticação e controle de acesso
  3. Ajustar as interfaces para corresponder às estruturas de dados do seu backend
  4. Adicionar tratamento de erros e feedback visual
